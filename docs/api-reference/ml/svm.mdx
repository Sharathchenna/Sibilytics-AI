## Overview

The SVM API provides endpoints for training Support Vector Machine classifiers with multiple kernels and grid search optimization.

---

## Upload Dataset

Upload a classification dataset.

<Card title="POST /api/svm/upload-dataset" horizontal />

### Request

```bash
curl -X POST https://api.sibilytics-ai.in/api/svm/upload-dataset \
  -F "file=@iris.csv"
```

### Response

```json
{
  "dataset_id": "svm_abc123",
  "filename": "iris.csv",
  "columns": ["sepal_length", "sepal_width", "petal_length", "petal_width", "species"],
  "rows": 150,
  "classes": ["setosa", "versicolor", "virginica"],
  "class_counts": {
    "setosa": 50,
    "versicolor": 50,
    "virginica": 50
  }
}
```

## Train SVM Models

Train SVM models with grid search cross-validation.

<Card title="POST /api/svm/train" horizontal />

### Request

```bash
curl -X POST https://api.sibilytics-ai.in/api/svm/train \
  -H "Content-Type: application/json" \
  -d '{
    "dataset_id": "svm_abc123",
    "feature_columns": [0, 1, 2, 3],
    "target_column": 4,
    "kernels": ["rbf", "linear", "poly", "sigmoid"],
    "c_values": [1, 2, 3, 4, 5, 6, 7, 8, 9],
    "gamma_values": [0.00001, 0.0001, 0.001, 0.01, 0.1],
    "cv_folds": 5,
    "test_sizes": [0.2, 0.3]
  }'
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `dataset_id` | string | Yes | Dataset ID from upload |
| `feature_columns` | array | Yes | Indices of feature columns |
| `target_column` | integer | Yes | Index of target column |
| `kernels` | array | Yes | Kernels to evaluate |
| `c_values` | array | Yes | C values for grid search |
| `gamma_values` | array | Yes | Gamma values for grid search |
| `cv_folds` | integer | Yes | Cross-validation folds |
| `test_sizes` | array | No | Test set sizes to evaluate |

### Response

```json
{
  "model_id": "model_xyz789",
  "results": {
    "rbf": {
      "best_params": {"C": 5, "gamma": 0.01},
      "accuracy": 0.9667,
      "precision": 0.9700,
      "recall": 0.9667,
      "f1_score": 0.9667,
      "auc": 0.9833
    },
    "linear": {
      "best_params": {"C": 3},
      "accuracy": 0.9333,
      "precision": 0.9400,
      "recall": 0.9333,
      "f1_score": 0.9333,
      "auc": 0.9611
    }
  },
  "best_kernel": "rbf",
  "training_time_seconds": 45.2
}
```

## Make Predictions

Use trained model to classify new data.

<Card title="POST /api/svm/predict" horizontal />

### Request

```bash
curl -X POST https://api.sibilytics-ai.in/api/svm/predict \
  -H "Content-Type: application/json" \
  -d '{
    "model_id": "model_xyz789",
    "kernel": "rbf",
    "features": [5.1, 3.5, 1.4, 0.2]
  }'
```

### Response

```json
{
  "prediction": "setosa",
  "probabilities": {
    "setosa": 0.95,
    "versicolor": 0.03,
    "virginica": 0.02
  },
  "confidence": 0.95
}
```

## Download Results

Download training results as Excel file.

<Card title="POST /api/svm/download-results" horizontal />

### Request

```bash
curl -X POST https://api.sibilytics-ai.in/api/svm/download-results \
  -H "Content-Type: application/json" \
  -d '{"model_id": "model_xyz789"}' \
  --output results.xlsx
```

## Download Plots

### Single Plot

<Card title="POST /api/svm/download-plot" horizontal />

```bash
curl -X POST https://api.sibilytics-ai.in/api/svm/download-plot \
  -H "Content-Type: application/json" \
  -d '{
    "model_id": "model_xyz789",
    "plot_type": "roc_curve",
    "kernel": "rbf"
  }' \
  --output roc_curve.png
```

Available plot types:
- `roc_curve` - ROC curve
- `confusion_matrix` - Confusion matrix
- `decision_boundary` - Decision boundary (2D only)

### All Plots (ZIP)

<Card title="POST /api/svm/download-all-plots" horizontal />

```bash
curl -X POST https://api.sibilytics-ai.in/api/svm/download-all-plots \
  -H "Content-Type: application/json" \
  -d '{"model_id": "model_xyz789"}' \
  --output all_plots.zip
```

## Supported Kernels

| Kernel | Parameter | Best For |
|--------|-----------|----------|
| `rbf` | C, gamma | Non-linear, most versatile |
| `linear` | C | Linearly separable data |
| `poly` | C, gamma, degree | Complex non-linear patterns |
| `sigmoid` | C, gamma | Neural network-like behavior |

---

## Grid Search Parameters

### C Values (Regularization)

- Low (1-3): Smoother boundary, better generalization
- High (7-9): Tighter fit, may overfit

### Gamma Values

- Low: Far-reaching influence, smoother
- High: Local influence, more complex

## Error Responses

### Dataset Not Found

```json
{
  "detail": "Dataset 'svm_abc123' not found or expired"
}
```

### Invalid Column Index

```json
{
  "detail": "Feature column 10 out of range"
}
```

### Training Error

```json
{
  "detail": "Training failed: insufficient samples for 5-fold CV"
}
```
