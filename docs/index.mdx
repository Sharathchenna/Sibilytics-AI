## What is Sibilytics AI?

**Sibilytics AI** is an advanced web-based platform designed for feature extraction from sensor and denoised signals, which can subsequently be used for machine learning-based analyses such as classification, clustering, and prediction.

## Key Capabilities

<AccordionGroup>
  <Accordion icon="bolt" title="High Performance">
    Processes signal files with **800,000+ data points in under 10 seconds** using the LTTB downsampling algorithm.
  </Accordion>
  <Accordion icon="file" title="Multi-format Support">
    Supports multiple file formats including `.txt`, `.lvm`, `.csv`, and `.xlsx` with auto-detection of headers and delimiters.
  </Accordion>
  <Accordion icon="wave-pulse" title="13 Wavelet Types">
    Implements 13 different Biorthogonal wavelet types with up to 20 decomposition levels.
  </Accordion>
  <Accordion icon="chart-line" title="Comprehensive Visualization">
    Generates time-domain, frequency-domain, spectrogram, and wavelet decomposition plots.
  </Accordion>
</AccordionGroup>
