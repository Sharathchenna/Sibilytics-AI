## Data Analysis

When you upload a dataset, Sibilytics AI automatically analyzes:

| Analysis | Description |
|----------|-------------|
| **Null Summary** | Count of null values per column |
| **Data Types** | Numeric vs categorical detection |
| **Sample Preview** | First 10 rows of data |
| **Basic Statistics** | Mean, median, std, min, max per column |

## Visualization Types

### Scatter Plot

Compare two numeric columns with interactive points.

- **X-Axis**: Select any numeric column
- **Y-Axis**: Select any numeric column
- **Color By**: Optional categorical column for grouping
- **Export**: Download as PNG

### Histogram

Analyze distribution of a single column with computed statistics.

- **Bins**: Configurable number of bins
- **Statistics**: Skewness and Kurtosis displayed
- **Density**: Optional kernel density overlay

### Correlation Heatmap

Visualize relationships between all numeric columns.

- **Method**: Pearson correlation
- **Scale**: -1 (negative) to +1 (positive)
- **Insight**: Identify highly correlated features

### 3D Surface Plot

Visualize relationships between three numeric columns.

- **X, Y, Z Axes**: Select three columns
- **Interactive**: Rotate and zoom the 3D plot
- **Use case**: Understanding multi-dimensional relationships

### Filtered Plot

View data within a specific interval.

- **Select column**: Choose the filtering column
- **Set interval**: Min and max values
- **Plot filtered**: Visualize the subset

## Null Value Handling

Multiple strategies available for handling missing values:

| Strategy | Description | Best For |
|----------|-------------|----------|
| **Replace with Mean** | Fill nulls with column average | Numeric, normal distribution |
| **Replace with Median** | Fill nulls with column median | Numeric, skewed distribution |
| **Replace with Mode** | Fill nulls with most frequent value | Categorical data |
| **Replace with Std Dev** | Fill nulls with standard deviation | Edge case handling |
| **Replace with Max/Min** | Fill nulls with extrema | Boundary value analysis |
| **Delete Rows** | Remove rows containing nulls | Small % of nulls |
| **Delete Column** | Remove entire column | High % of nulls |

<Warning>
  Deleting rows or columns permanently modifies your working dataset. Download the original first if needed.
</Warning>

## Categorical Encoding

Transform categorical (text) data into numeric format for machine learning.

### Label Encoding

Assigns integer values to each category.

```
Category → Number
Apple    → 0
Banana   → 1
Cherry   → 2
```

**Best for**: Ordinal data (has natural order like Small < Medium < Large)

### One-Hot Encoding

Creates binary columns for each category.

```
Fruit   →   Fruit_Apple  Fruit_Banana  Fruit_Cherry
Apple   →       1            0             0
Banana  →       0            1             0
Cherry  →       0            0             1
```

**Best for**: Nominal data (no order) with few unique values

### Frequency Encoding

Replaces categories with their frequency in the dataset.

```
Category (freq) → Number
Apple (30%)     → 0.30
Banana (50%)    → 0.50
Cherry (20%)    → 0.20
```

**Best for**: High cardinality categorical features
